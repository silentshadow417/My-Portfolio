Project-1: Number Guessing Game












This is an interactive console game built using C programming. The system automatically generates a unique random number for each session. Players have to guess the correct number to win. It provides real-time hints like "Too High" or "Too Low" after every guess. A counter variable tracks and displays the total attempts taken by the player. This project helped me master while loops and if-else logic building.

Project-2: Quize Game











This is a terminal-based dynamic quiz application developed in C. It displays multiple-choice questions with clear options (A, B, C, D). The application processes user inputs efficiently using switch-case statements. I used modular programming by splitting the logic into clean custom functions. The program tracks scores dynamically and shows a final score report at the end. It includes basic input validation to handle wrong keystrokes smoothly. This project strengthened my understanding of functions, loops, and data tracking.
